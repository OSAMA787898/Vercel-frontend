# Realtime Chat Application 🚀

A real-time chat application where multiple users can create or join chat rooms and communicate instantly.

## 🌐 Live Demo
https://vercel-frontend-iota-dun.vercel.app

## 🛠 Technologies Used
- React.js
- Node.js
- Express.js
- Socket.io
- Tailwind CSS
- JavaScript
- Git & GitHub
- Vercel (Frontend Deployment)
- Render (Backend Deployment)

## ✨ Features
- Create or join chat rooms
- Share room link with friends
- Real-time messaging
- Typing indicator
- Emoji support
- Online users list
- Modern animated UI
